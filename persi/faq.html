Objectives

  • Why are you creating this Katka Project?

    Katka project is intended to keep materials from my learning exercises. Over the many years, I had built software for windows, servers, and Azure cloud. Now as I am exploring open source software, I keep this as a learning base to keep track of stuff I am learning and experimenting.

  • What is Persi? Why?

    Everyone has peronal information that they collect over time. I am expecting to have a simpe web page that allows me to track my personal destinations. This web page will be independent of the underlying browser or operating system, so I can access and use this from any device or browser.

  • When will Persi be real?

    There is no definite end date. This is an continuously evolving project. I work on this project a few minutes each day to explore different stuff. It is amazing to see how much information and tools are available online!

More Information

  • Where can I find some tutorials?

    Check out the git project at Project Katka for a collection of tutorial links. Of course searching on the internet will also provide you more pointers ... though the information is often scattered.

Contributions
